# Car Sales Analysis and Prediction

Logistic regression model predicting car purchase likelihood, with cross-market forecasting and a Tableau dashboard.

## Stack
- Python (pandas, scikit-learn, seaborn, matplotlib)
- Logistic Regression
- Tableau

## Structure
- `car_analysis.py` — main analysis & model script
- `dataset/` — input and output data files
- `Presentation/` — Tableau workbook and dashboard exports
